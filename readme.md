## Professional Care for Your Screen Reader
Is your NVDA feeling sluggish? Are strange errors popping up after installing new add-ons? **DoctorNVDA** acts as a personal physician for your software. It provides high-precision tools to diagnose conflicts and restore health instantly.

> **Main Menu Shortcut:** Press **Alt + Windows + D**

### Key Features

* **NVDA Version Control:** Get exact build info and copy it to clipboard instantly.
* **Create & Restore Settings:** Create "Point-in-Time" recovery points for your profiles and dictionaries.
* **Safe Mode Restart:** Verify issues by restarting with all add-ons disabled.
* **Binary Search Debugging:** Automatically isolate problematic add-ons by splitting your collection in half until the culprit is found.
* **Direct Config Access:** One-click access to `%APPDATA%\nvda`.
* **RAM Diagnostic:** Quick hardware scan to copy RAM specs for support tickets.
