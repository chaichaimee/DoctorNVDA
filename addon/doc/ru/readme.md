## Профессиональная забота о вашем скринридере
NVDA стал работать медленнее? Появились странные ошибки после установки дополнений? **DoctorNVDA** действует как личный врач для вашего ПО, предоставляя инструменты для мгновенного восстановления работоспособности.

> [!IMPORTANT]
> **Горячая клавиша меню:** Нажмите **Alt + Windows + D**

## Подробное описание функций

### • Создание и восстановление настроек
Функция **"Точка восстановления"** позволяет сохранить идеальную конфигурацию и вернуться к ней в любой момент.

> **Процесс безопасности:**
> 1. Создайте точку восстановления стабильных настроек.
> 2. Если возникла проблема, выберите дату из меню.
> 3. DoctorNVDA автоматически вернет настройки и перезапустит программу.

### • Диагностика бинарным поиском (Детектив дополнений)
Автоматизированный процесс **бинарного поиска** позволяет быстро найти конфликтующее дополнение среди десятков других, разделяя список пополам на каждом этапе.

### • Проверка оперативной памяти
Мгновенное получение данных о RAM (производитель, объем, тип, скорость) с автоматическим копированием в буфер обмена.

> [!TIP]
> **Рекомендация врача:** Всегда создавайте точку восстановления перед установкой новых крупных дополнений или внесением изменений.
