## Профессиональный уход за вашей программой чтения с экрана

<br>

NVDA работает медленно? **DoctorNVDA** — это личный врач для вашего ПО, предоставляющий инструменты для мгновенного восстановления работоспособности.

<br>

> **Быстрый доступ к меню:** Нажмите **Alt + Windows + D**

<br>

### • Диагностика бинарным поиском
<br>
**Проверка здоровья:** Диалог спрашивает: **"Проблема ИСЧЕЗЛА?"**
- **ДА:** Виновник находится среди отключенных дополнений.
- **НЕТ:** Виновник находится в группе, которая осталась активной.
