## Профессиональная забота о вашем экранном дикторе
Ваш NVDA работает медленно? **DoctorNVDA** действует как личный врач для вашего ПО, предоставляя инструменты для мгновенного восстановления здоровья системы.

> **Горячая клавиша меню:** Нажмите **Alt + Windows + D**

### Основные возможности

* **Контроль версии:** Точная информация о сборке в один клик.
* **Точки восстановления:** Создание резервных копий профилей и словарей.
* **Безопасный режим:** Перезапуск без дополнений для выявления ошибок.
* **Бинарный поиск:** Автоматический поиск конфликтующего дополнения методом деления списка пополам.
* **Прямой доступ:** Мгновенное открытие папки настроек пользователя.
