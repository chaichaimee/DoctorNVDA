## Des soins professionnels pour votre lecteur d'écran
Votre NVDA semble lent ? **DoctorNVDA** agit comme un médecin personnel pour votre logiciel, offrant des outils de haute précision pour diagnostiquer les conflits instantanément.

> **Raccourci du menu principal :** Appuyez sur **Alt + Windows + D**

### Caractéristiques principales

* **Contrôle de version :** Copiez les infos de build exactes dans le presse-papiers.
* **Sauvegarde & Restauration :** Créez des points de récupération "Point-in-Time".
* **Mode sans échec :** Redémarrez sans extensions pour isoler les problèmes.
* **Recherche binaire :** Identifiez l'extension coupable automatiquement en divisant votre collection par deux.
* **Accès direct :** Accédez au dossier de configuration en un clic.
