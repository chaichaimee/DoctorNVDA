## Soins Professionnels pour votre Lecteur d'Écran

<br>

Votre NVDA est lent ? **DoctorNVDA** agit comme un médecin personnel pour votre logiciel, offrant des outils de haute précision.

<br>

> **Raccourci du Menu Principal :** Appuyez sur **Alt + Windows + D**

<br>

### • Débogage par Recherche Binaire
<br>
**Bilan de santé :** Un dialogue demande : **"Le problème a-t-il DISPARU ?"**
- **OUI :** Le coupable est dans le groupe désactivé.
- **NON :** Le coupable est dans le groupe resté actif.
