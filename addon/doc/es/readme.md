## <p align="center">Descripción</p>

<br>

¿Sientes que tu NVDA está lento? ¿Aparecen errores extraños después de instalar nuevos complementos? **DoctorNVDA** actúa como un médico personal para tu software. Proporciona herramientas de alta precisión para diagnosticar conflictos y restaurar la salud al instante, asegurando que nunca pierdas tu configuración ni tu tiempo.

<br>

Este complemento está diseñado para todos los usuarios, independientemente de su habilidad técnica. Simplifica la resolución de problemas complejos en unos pocos clics, permitiéndote gestionar copias de seguridad, identificar complementos problemáticos y realizar reinicios de emergencia sin necesidad de tocar archivos complicados del sistema o el Administrador de tareas de Windows.

<br>

## <p align="center">Teclas Rápidas</p>

<br>

DoctorNVDA utiliza un sistema inteligente de **Pulsación Múltiple**. Esto significa que solo necesitas recordar una combinación de teclas principal para realizar múltiples acciones de recuperación.

<br>

> **Comando Principal: ALT + Windows + D**
>
> *   **Una Pulsación:** Abre el **Menú Principal de DoctorNVDA** para acceder a todas las funciones.
>
> *   **Doble Pulsación:** **Reinicia NVDA** instantáneamente (Modo Normal).
>
> *   **Triple Pulsación:** **Reinicio de Emergencia con Complementos Desactivados** (Modo Seguro).

<br>

## <p align="center">Características</p>

<br>

### 1. Depuración por Búsqueda Binaria (El Detective de Complementos)

<br>

Encontrar un solo complemento roto entre docenas es como buscar una aguja en un pajar. Esta función automatiza el método de "Búsqueda Binaria" para encontrar al culpable por ti.

<br>

**Guía Paso a Paso:**
1. Inicia el diagnóstico desde el menú de DoctorNVDA.
2. NVDA se reiniciará con la mitad de tus complementos desactivados.
3. Un diálogo preguntará: **"¿Ha DESAPARECIDO el problema?"**
4. Si respondes **Sí**, el detective sabe que el problema está en la mitad desactivada. Si es **No**, está en la mitad activa.
5. El proceso se repite, reduciendo el grupo hasta que el complemento problemático exacto queda aislado.

<br>

### 2. Crear y Restaurar Ajustes de NVDA (Máquina del Tiempo)

<br>

Protege tus diccionarios personalizados, gestos y perfiles. Esta función crea un **"Punto de Recuperación"** al que puedes volver en cualquier momento.

<br>

**Guía Paso a Paso:**
1. **Para Respaldar:** Selecciona "Crear Recuperación" cuando tu NVDA funcione perfectamente.
2. **Para Restaurar:** Si tus ajustes se desconfiguran, selecciona "Restaurar Ajuste de NVDA" en el menú.
3. Elige una fecha de recuperación de la lista.
4. DoctorNVDA reemplazará automáticamente los archivos dañados y reiniciará NVDA por ti.

<br>

### 3. Resumen de Información del Sistema

<br>

¿Necesitas reportar un error o verificar la salud de tu sistema? Esta herramienta reúne todas las estadísticas vitales en un solo lugar, sin necesidad de conocimientos técnicos.

<br>

**Guía Paso a Paso:**
1. Selecciona "Resumen de Info del Sistema" en el menú.
2. Aparecerá un informe limpio mostrando tu versión de NVDA, versión de Windows y arquitectura del sistema.
3. Esta info se formatea automáticamente para que sea fácil de leer y compartir con desarrolladores o equipos de soporte.

<br>

### 4. Acceso Instantáneo a la Configuración de Usuario

<br>

<br>

%APPDATA%\nvda
