## Cuidado Profesional para tu Lector de Pantalla

<br>

¿Tu NVDA se siente lento? **DoctorNVDA** actúa como un médico personal para tu software, ofreciendo herramientas de alta precisión para restaurar la salud al instante.

<br>

> **Atajo del Menú Principal:** Presiona **Alt + Windows + D**

<br>

### • Depuración por Búsqueda Binaria
<br>
**Ejemplo: 1 culpable entre 40 complementos**
- **Chequeo de Salud:** Un diálogo pregunta: **"¿Se ha SOLUCIONADO el problema?"**
    - **SÍ:** El culpable está en el grupo deshabilitado.
    - **NO:** El culpable está en el grupo activo.
