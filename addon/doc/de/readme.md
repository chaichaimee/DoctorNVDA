## Professionelle Pflege für Ihren Screenreader
Fühlt sich Ihr NVDA träge an? Treten nach der Installation neuer Add-ons seltsame Fehler auf? **DoctorNVDA** fungiert als persönlicher Arzt für Ihre Software. Er bietet hochpräzise Werkzeuge zur Diagnose von Konflikten.

> **Hauptmenü-Tastenkombination:** Drücken Sie **Alt + Windows + D**

### Hauptfunktionen

* **Versionskontrolle:** Exakte Build-Informationen sofort in die Zwischenablage kopieren.
* **Sichern & Wiederherstellen:** Erstellen Sie "Punkt-zu-Zeit"-Wiederherstellungspunkte für Ihre Konfiguration.
* **Abgesicherter Modus:** Starten Sie NVDA ohne Add-ons neu, um Fehler zu isolieren.
* **Binäre Suche (Detektiv):** Finden Sie das fehlerhafte Add-on automatisch durch systematisches Halbieren der Liste.
* **Direkter Zugriff:** Öffnet den Benutzerkonfigurationsordner mit einem Klick.
